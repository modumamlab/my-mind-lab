- 예약 필수 동의 3개 직접 체크 + 전자서명 입력 후에만 예약 신청 가능
- 미체크/서명 누락 시 안내 팝업과 오류 메시지 표시

## V16 - 프로그램 가격 및 섹션 순서 수정
- 부부 마음이음/부모-자녀 마음이음 가격 계산을 기본검사 2개 80,000원 + 상담비 + 유료 추가검사로 수정
- 대면/찾아가는 상담 80,000원, 비대면 상담 50,000원, 추가검사 1건 30,000원 규칙 반영
- 마음 검사 카탈로그와 모두맘 프로그램 섹션 순서 교체


## V20

- 상담예약 명칭을 심리검사 예약으로 변경

- 가격 계산 정책을 상담비(기본검사 포함) + 기본검사 추가 + 추가검사로 정리


## V21 랜딩 흐름 정리
- 페이지 순서: 부호 질문 → 네 개의 부호 → 내 마음을 부탁해 → 심리검사 → AI 접수면접 → 모두맘 프로그램 → 심리검사 예약
- 메뉴: 연구소 소개 / AI 마음리포트 / 심리검사 / AI 접수면접 / 심리검사 예약 / 마이페이지
- AI 접수면접 페이지의 “AI 마음리포트 예시”를 “AI 마음지기 보고서 예시”로 수정
- 전문가 상담 연결 문구를 임상심리 전문가의 상담 준비자료 활용 안내로 수정

## V22 - AI 마음지기 안내 및 검사 설명 팝업
- AI 접수면접 페이지의 `AI 마음지기 보고서 예시` 문구를 `AI 마음지기를 만나면`으로 수정했습니다.
- 안내 카드 내용을 `마음 정리를 돕습니다 / 필요한 검사를 추천합니다 / 심리검사 상담에 도움이 됩니다` 방향으로 정리했습니다.
- 추천 검사 조합에서 검사명을 클릭하면 검사 설명 팝업이 열리도록 추가했습니다.
- 검사 설명 팝업에는 `무엇을 알 수 있나요?`, `상담에서는 어떻게 활용되나요?` 항목이 표시됩니다.


## V23 - 결제 예정 금액 비대면 계산 수정
- 비대면 화상/전화/문자 선택 시 `비대면` 안의 `대면` 글자 때문에 80,000원으로 계산되던 오류 수정
- 대면은 80,000원, 비대면은 50,000원으로 즉시 반영되도록 계산 조건 정리
