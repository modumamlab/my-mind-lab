# 모두의 마음연구소 상담 준비 서류

이 폴더는 홈페이지에서 직접 연결되는 상담 준비 서류 보관 위치입니다.

- application.pdf: 상담신청서 PDF
- application.docx: 상담신청서 Word 원본
- consent.pdf: 개인정보 수집·이용 및 심리검사/상담 동의서 PDF
- consent.docx: 동의서 Word 원본
- index.html: 내담자에게 발송할 수 있는 서류 안내 페이지

관리자 문자/카카오톡 안내 문구는 `admin/js/admin.js`의 `copyDocumentReminder`에서 수정합니다.
