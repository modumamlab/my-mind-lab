# AI 심리검사 추천 매트릭스 v1.0

## 추천 원칙
AI는 검사명을 먼저 제시하지 않는다.  
먼저 평가 목적을 설명한 뒤, 그 목적에 맞는 검사를 추천한다.  
검사는 마음사랑과 인싸이트에서 취급하는 검사를 중심으로 제안한다.

## 1. 기질·성격 이해
추천 검사:
- TCI 기질 및 성격검사
- STS 기질검사

추천 상황:
- 반복되는 선택 패턴
- 대인관계 반복 문제
- 스트레스 반응 이해
- 자기이해 욕구

## 2. 우울·불안
추천 검사:
- 우울검사
- 불안검사
- 필요 시 MMPI-2

추천 상황:
- 무기력
- 불면
- 걱정 증가
- 신체 긴장
- 정서 조절 어려움

## 3. 직장 스트레스
추천 검사:
- 직무스트레스검사
- TCI 기질 및 성격검사

추천 상황:
- 직장 내 갈등
- 소진
- 상사 또는 동료와의 갈등
- 업무 부담

## 4. 대인관계
추천 검사:
- TCI
- 대인관계문제검사
- SCT 문장완성검사

추천 상황:
- 반복되는 갈등
- 거절 민감성
- 관계 회피
- 표현 어려움

## 5. 부부·커플
추천 검사:
- 부부 TCI

추천 상황:
- 반복되는 갈등
- 의사소통 어려움
- 서로의 차이를 이해하고 싶은 경우

## 6. 부모-자녀
추천 검사:
- PAT 부모양육태도검사
- K-CDI 아동발달검사
- 필요 시 부모 TCI

추천 상황:
- 아이 행동 이해
- 양육 스트레스
- 부모-자녀 상호작용 이해

## 7. 진로
추천 검사:
- Holland 직업흥미검사
- 직업가치관검사
- TCI

추천 상황:
- 진로 방향 혼란
- 적성·흥미 탐색
- 직업 선택 고민

## 관리자 전용 추가 항목

- 추가 고려 검사
- 검사 우선순위
- 상담자가 최종 선택할 항목
